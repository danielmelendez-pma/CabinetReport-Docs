# Política de Privacidad y Protección de Datos

Cabinet Report respeta y protege la privacidad de sus usuarios conforme a la **Ley 81 de 26 de marzo de 2019**, sobre Protección de Datos Personales de la República de Panamá.

Esta Política de Privacidad complementa el **Código de Ética y Confidencialidad** y los **Términos y Condiciones de Uso**.

### 1. Datos recopilados

Cabinet Report podrá gestionar datos proporcionados directamente por el usuario, tales como:

* Información de identificación empresarial o personal
* Datos técnicos de proyectos y producción
* Información operativa ingresada voluntariamente en el sistema

Cabinet Report **no recopila datos sin conocimiento del usuario**.

### 2. Finalidad del uso de los datos

Los datos se utilizan exclusivamente para:

* El funcionamiento correcto del software
* La prestación del servicio contratado
* Soporte técnico solicitado por el usuario

No se utilizan los datos con fines publicitarios, estadísticos externos ni comerciales.

### 3. Confidencialidad y seguridad

Los datos se manejan bajo principios de:

* Confidencialidad
* Acceso limitado
* Uso exclusivo por el usuario

Cabinet Report no vende, cede ni transfiere información a terceros.

### 4. Derechos del titular de los datos

El usuario puede, conforme a la Ley 81:

* Acceder a su información
* Solicitar corrección
* Solicitar eliminación
* Limitar el uso de sus datos

Estas solicitudes deberán realizarse por los canales oficiales de soporte.

### 5. Conservación de la información

Los datos se conservan únicamente durante el tiempo necesario para la prestación del servicio o mientras exista una relación activa con el usuario.

### 6. Aceptación

El uso del software implica la aceptación expresa de esta Política de Privacidad.
