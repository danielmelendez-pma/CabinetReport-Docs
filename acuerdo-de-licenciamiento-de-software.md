# Acuerdo de Licenciamiento de Software

El presente Acuerdo regula el uso del software Cabinet Report conforme al plan de suscripción contratado.

### 1. Naturaleza de la licencia

Cabinet Report se concede bajo una **licencia de uso**:

* No exclusiva
* No transferible
* Limitada
* Revocable

La licencia **no constituye venta del software**.

### 2. Alcance por plan

Cada plan de suscripción define:

* Número de usuarios permitidos
* Funcionalidades habilitadas
* Límites operativos (reportes, almacenamiento, modificaciones, etc.)

El uso fuera de los límites del plan podrá derivar en suspensión del servicio.

### 3. Restricciones

Queda prohibido:

* Compartir credenciales
* Reproducir o redistribuir el software
* Modificar, descompilar o intervenir el sistema
* Utilizar el software para fines ilícitos

### 4. Propiedad intelectual

Todo el software, lógica, diseño, documentación y algoritmos son propiedad intelectual del desarrollador, protegidos por las leyes de Panamá.

### 5. Terminación de la licencia

La licencia podrá ser suspendida o revocada por:

* Incumplimiento contractual
* Uso indebido del sistema
* Violación de derechos de propiedad intelectual
